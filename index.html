Hello World

To get started we will do the classic hello world example.

Open the web console by pressing ctrl+shift+k and type in the code below and hit enter.

console.log('hello world');
Variables

Variables allow you store a value and give it a name.

Once a variable has been defined it can be used and even changed later on throughout the code.

let age = 20;
Constants

Constants are similar to variables, they allow you to store a value and give it a name.

The main difference between a variable and a constant is you can not change the value of a constant unlike a variable.

const name = john;
If...Else

If...Else statements are known as conditionals, they allow you to make decisions in your code.

Run 'x' if the given condition is true, else run 'y' if condition is false.

let color = "blue"; if (color === "blue") { console.log("color is blue"); } else { console.log("color is not blue"); }
Functions

Functions are reusable blocks of code.

They can help eliminate repetitive code, which decreases the size and makes the code cleaner and more readable.

function greet() { console.log("hello"); } greet();
Data Types

There are several different data types in programming languages.

Data types define the type of data things are so they can be interpreted properly.

  • Number - simple numbers
  • Boolean - true or false
  • String - text and characters
  • Undefined - undefined value
  • null - null value
